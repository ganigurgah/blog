# Kod Dünyasına Hoş Geldiniz 📝

Bu blog, **GitHub Pages** üzerinde çalışan ve dinamik içerik yönetimini tamamen istemci taraflı (client-side) halleden özel bir yapıya sahiptir.

## Öne Çıkan Özellikler
* **Hafif ve Hızlı:** Sunucu taraflı hiçbir derleme veya backend motoru barındırmaz.
* **Markdown Desteği:** `marked.js` kütüphanesi sayesinde tüm `.md` dökümanları anında HTML'e parse edilir.
* **Dinamik Tarama:** Yeni bir not ekleyip pushlamanız, listenin güncellenmesi için yeterlidir.

### Örnek Kod Bloğu (Java)
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Statik blogdan selamlar!");
    }
}
```
